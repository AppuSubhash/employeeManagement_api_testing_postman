# Employee Management API Testing using Postman

## Project Overview

This project demonstrates REST API testing using Postman by performing CRUD operations on an Employee Management API.

## APIs Covered

- GET Employee
- POST Employee
- PUT Employee
- DELETE Employee

## Test Scenarios

### Positive Testing

- Successful GET Request
- Successful POST Request
- Successful PUT Request
- Successful DELETE Request

### Negative Testing

- Invalid Endpoint
- Invalid Resource ID

### Validations Performed

- Status Code Validation
- Response Body Validation
- Response Header Validation
- Response Time Validation

## Tools Used

- Postman
- Collection Runner
- GitHub
